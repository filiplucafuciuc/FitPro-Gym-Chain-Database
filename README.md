**FitPro** is an Oracle Database project designed for managing a **real-world gym chain**.
The database models clients, gyms, trainers, courses, equipment, subscriptions, schedules,
personal training sessions, and client feedback.

This project focuses on **practical database design**, integrity constraints, and
realistic SQL queries used in production-like systems.

---

## Project Overview
The database supports:
- management of gym locations and cities
- tracking gym equipment and its condition
- client registration and subscription history
- course scheduling and participation
- personal training sessions
- trainer evaluation and client feedback

The schema is designed to reflect **real operational needs** of a fitness business.
